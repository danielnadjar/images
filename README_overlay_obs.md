# Overlay OBS UMAIN (1920x1080)

Fichier principal : `overlay_obs_umain.html`

## Utilisation rapide dans OBS
1. Ajoute une **Source Navigateur**.
2. Coche **Fichier local** et sélectionne `overlay_obs_umain.html`.
3. Mets la résolution à **1920 x 1080**.
4. Coche **Actualiser le navigateur quand la scène devient active**.
5. Place ta source **Caméra** sous l'overlay puis redimensionne-la pour remplir le cercle central.

## Personnalisation
- Modifier le logo : remplacer `client mystere doc.png` par ton fichier et ajuster la classe `.umain-logo`.
- Modifier le bandeau des outils IA : éditer le texte dans `.ai-marquee span`.
- Ajuster la taille du placeholder caméra : classe `.camera-frame` (width/height).
